# Finite-TNG

Welcome to the new official Website for Finite Research Group! This site is built using Astro, Vue, and React to provide a modern and efficient browsing experience. Here, you can explore our latest research, projects, Store, and more.


Finite Research Group is dedicated to pushing the boundaries of technology and innovation. Our team of experts works on cutting-edge projects in various fields, including programming languages, advanced materials, artificial intelligence, and computing platforms.

## Contributing
We welcome contributions from the community! If you'd like to contribute, please fork the repository and submit a pull request with your changes. Make sure to follow our coding standards and include relevant tests.
